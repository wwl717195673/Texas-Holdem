# 开发指南
这篇文档描述了claude code 工作时，应该遵守的流程
## 开发 SOP
1. 了解用户需求
2. 分析用户需求
3. 进行代码实现
4. 完善严谨的单元测试
## 开发注意事项
1. 每一个方法都加上方法级别的中文注释
2. 当一个方法超过 30 行代码时，给这个方法的关键节点也加上中文注释
3. 每次代码实现都生成 1 个实现文档，写明本次生成做了什么 怎么做的,文件名为本次实现的基本功能 简短一点 markdown格式 放在/Users/wilenwang/works/ai/Texas-Holdem/implement_doc 目录下
4. 当项目有大量代码开发需求（新增多个文件、涉及多个模块、实现完整功能等）时，必须生成完整的实现文档，详细记录每个模块的实现细节、代码结构和使用方法
